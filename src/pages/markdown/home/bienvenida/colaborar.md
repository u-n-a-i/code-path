# Cómo colaborar

Este proyecto está hecho para crecer con aportes. Si quieres colaborar, sigue estos pasos:

1. Clona el repositorio: `git clone <repo>`.
2. Crea una rama con un nombre claro: `git checkout -b mejora-markdown`.
3. Añade o mejora contenido en `src/pages/markdown` o en `src/data`.
4. Prueba localmente: `pnpm install && pnpm dev`.
5. Haz un pull request explicando el cambio.

## Tipos de colaboración

- Añadir nuevos artículos teóricos.
- Mejorar redacción y ejemplos.
- Añadir componentes UI para navegación o estilo.
- Corregir errores de contenido o de rutas.

## Normas básicas

- Mantén cada tema separado en su propia carpeta y archivo.
- Usa nombres de archivo descriptivos (por ejemplo: `cpu.md`, `funciones.md`).
- Incluye siempre un párrafo introductorio y, si aplica, ejemplos.

> Contribuir no solo mejora el proyecto: te ayuda a reforzar tus propios conocimientos.
