# Propósito del proyecto

<img src="/img/header.png" alt="Code Path">

Este proyecto integra las rutas ASIR, DAM y DAW bajo un enfoque común: el desarrollo de software.


## ¿Qué busca lograr?

- Centralizar el aprendizaje técnico en un solo lugar.
- Ofrecer contenidos que vayan de lo esencial a lo avanzado (aprendizaje progresivo).
- Hacer accesible la teoría y la práctica para estudiantes de ciclos formativos y grado.

## ¿Para quién es?

- Estudiantes de ASIR, DAM, DAW y similar.
- Desarrolladores que necesitan recursos rápidos.
- Personas que quieren aprender el ciclo de vida del desarrollo de software.

## ¿Qué incluye?

- Apuntes estructurados por tema (hardware, software, redes, web, datos).
- Recursos prácticos y ejemplos de código.
- Contenido pensado para estudiar por bloques y repasar de forma guiada.


> Este proyecto es una herramienta de estudio: no pretende sustituir un curso, sino servir de apoyo con apuntes claros, enlaces útiles y contenido fácilmente navegable.
